# Kotlin Spring Example


This is a CRUD project in Kotlin and Spring that provides a RESTful API to create, read,
update and delete data through HTTP requests. The API follows best practices recommended by the community
Kotlin and Spring and is documented with Swagger for easy viewing and testing of endpoints.


## Technologies used

- Kotlin 1.7.22
- Spring 3.0.4
- Swagger 3.0.0

## Run the Project

To run the project, just follow these steps:

1. Clone the project on your local machine;
2. Open the project in your preferred IDE and compile it;
3. Run the project in your IDE or via the command line;
4. Access the API documentation generated by Swagger to view and test available endpoints.

## Using

1. Start the application server
2. Access the Swagger documentation at `http://localhost:8080/swagger-ui.html`
3. Use CRUD operations to interact with the system
